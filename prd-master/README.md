# PRD MASTER - 顶级产品需求文档生成专家

## 项目简介

PRD MASTER 是一个专业的产品需求文档生成技能，用于创建企业级完整的 PRD 文档，覆盖从产品战略、业务流程、信息架构、功能设计、数据结构、接口规范到运营规则的完整体系。

输出文档可直接用于：
- 产品评审
- 研发开发
- 测试用例设计
- UI设计
- 项目管理

## 核心功能

- **完整PRD生成**：自动生成包含21个章节的完整产品需求文档
- **信息架构设计**：提供系统架构、页面结构和导航结构的设计
- **业务流程建模**：使用PlantUML生成业务流程图
- **状态机设计**：明确定义业务状态流转
- **数据模型设计**：规范数据实体和关系
- **API接口规范**：生成标准的API接口文档
- **交互规则定义**：提供完整的生命周期交互规范
- **技术架构规划**：推荐适合的技术栈
- **项目管理支持**：包含项目计划和风险评估

## 使用指南

### 生成完整PRD

当需要生成完整的PRD文档时，按照以下步骤操作：

1. **收集需求信息**：准备项目的基本信息
   - 项目名称和背景
   - 主要功能模块
   - 目标用户群体
   - 技术要求

2. **使用技能**：调用PRD MASTER技能，提供项目信息

3. **获取输出**：获得完整的PRD文档，包括：
   - 信息架构
   - 业务流程
   - 功能设计
   - 数据模型
   - API接口
   - 交互规范

### 生成特定模块

当只需要生成特定功能模块的PRD时：

1. **确认模块名称**：明确需要详细说明的功能模块

2. **使用技能**：指定模块名称，调用PRD MASTER技能

3. **获取输出**：获得包含以下内容的模块文档：
   - 功能描述
   - 核心功能列表
   - 用户场景
   - 页面设计
   - 数据结构

### 优化现有PRD

当需要优化现有PRD文档时：

1. **提供现有文档**：分享现有的PRD文档

2. **使用技能**：请求PRD MASTER对文档进行优化

3. **获取建议**：获得具体的改进建议和修改方案
